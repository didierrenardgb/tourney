# Tourney runner

```
usage: tourney.py [-h] [-pc PENALTY_COUNT] [-s SPEED] [players [players ...]]
    or
python -m tourney [-h] [-pc PENALTY_COUNT] [-s SPEED] [players [players ...]]

positional arguments:
  players               A space-separated list of participants for the tourney

optional arguments:
  -h, --help            show this help message and exit
  -pc PENALTY_COUNT, --penalty-count PENALTY_COUNT
                        Total amount of shots to be made by players; defaults
                        to 6(3 penalties each player)
  -s SPEED, --speed SPEED
                        Modifier for simulation run speed; defaults to 1.0
```
